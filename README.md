# Nexus

## Identidade Visual (UI/UX)

O nome "Nexus"sugere conexão e centralidade. Para isso, a identidade visual deve refletir esses conceitos, utilizando conceitos de **Glassmorphism** com um toque de **Dark Mode** industrial.

> **Glassmorphism:** é um estilo de design de interface que simula o efeito de vidro fosco, caracterizado pela translucidez, desfoque de fundo, bordas claras e cores vivas

### Paleta de Cores:
- **Deep Charcoal:** #0F172A
- **Electric Indigo:** #6366F1
- **Cyan:** #22D3EE
- **Slate Gray:** #94A3B8

### Tipografia:
- **Fonte Principal:** Inter
- **Fonte Secundária:** Roboto Mono

## ✨ Principais Funcionalidades

- **Gestão Ágil:** Criação e monitoramento de tarefas com estados dinâmicos.
- **Real-time Updates:** Sincronização instantânea entre dispositivos (WebSockets).
- **Métricas de Produtividade:** Dashboard com gráficos de conclusão de tasks.
- **Nexus CLI:** Interface de linha de comando para desenvolvedores "power users".

## Arquitetura

O projeto será desenvolvido com a arquitetura **Clean Architecture**, seguindo os princípios SOLID.

## Tecnologias

- **Frontend:** React + Tailwind CSS + TypeScript
- **Backend:** Go
- **Database:** PostgreSQL + Redis
- **Containerization:** Docker
